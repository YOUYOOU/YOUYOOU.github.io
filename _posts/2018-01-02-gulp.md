##环境依赖  
[node](http://nodejs.org)(v5.5.0)  
[npm](https://www.npmjs.com/)(v3.3.12)     

##安装构建工具[gulp](https://github.com/gulpjs/gulp)(v3.9.0)  
```
`$ npm install -g gulp`  ```  

##安装依赖管理系统[bower](http://bower.io/)(v1.7.7)  
```
`$ npm install -g bower` ```   

##安装依赖包  

```$ npm install
$ bower install```  

##浏览器中运行    
```  `$ gulp serve````  

## Ubuntu下所有的快捷方式都在：
```js
/usr/share/applications
```
